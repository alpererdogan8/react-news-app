# React.JS with NewsAPI

A simple news viewing interface

- Used Libraries
  - [Chakra UI](https://chakra-ui.com/)
  - [Keen-Slider](https://keen-slider.io/)
  - [Moment.JS](https://momentjs.com/)

- Used API
  - [NewsAPI](https://newsapi.org/)

Install dependencies\

`npm i`\
\
Build the project\

`npm start`
